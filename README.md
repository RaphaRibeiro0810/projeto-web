# projeto-web
Feito em linguagem HTML, CSS, JavaScipt no segundo semestre do curso, 2022. Este projeto teve como objetivo desenvolver um web site de simulação de compra e recarga de bilhetes de ônibus. Cada "cartão" possuía um código que seria utilizado para verificar seu registro no banco de dados e relacionar as recargas feitas com o respectivo cartão.
